## Requirements
* GNU Stow
* Git
* i3wm
* Polybar
* Dunst
* Rofi
* Picom
* Feh
* Spicetify
* Ranger
* Starship
* Maim


## GTK3 theme
* [DarK](https://www.gnome-look.org/p/1199936/)

## Installation

```
stow -t ~ bash config fonts git vim
sh ~/.vim/setup.sh
```
